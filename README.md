## 使用方法

首先克隆仓库代码：

```sh
git clone https://github.com/kuizuo/blog.git
```

安装依赖：

```sh
cd blog
npm install
```

启动项目：

```sh
npm start
```

构建项目：

```sh
npm build
```

关于部署，由于不同的服务器、平台有不同的方式，可以使用 Github Pages、FTP、自行搭建服务器等。
